NET3 Services:

Requisitos funcionais:
- O sistema permitirá fazer a busca e visualização de usuários;
- O sistema permitirá fazer a contratação de um serviço;
- O sistema permitirá fazer o cadastro, a edição e a exclusão de usuários;
- O sistema permitirá fazer a busca e visualização de serviços;
- O Sistema permitirá fazer o cadastro, a edição, a edição e a exclusão de serviços;
- O sistema deverá fazer a autenticação de usuários;
- O sistema permitirá aos visitantes fazerem o cadastro no sistema;
- O sistema permitirá aos usuários de fazerem uma denúncia em relação a outros usuários.
- O sistema permitirá o usuário de pedir um orçamento a um outro usuário antes de contratar o serviço;
- A cada pedido de contratação o sistema verifica se os dados dos dois usuários, tanto o contrante como o contratado, estão corretos.

Requisitos não-funcionais:
- O sistema permitirá aos usuários comuns de visualizarem informações de outros usuários comuns;
- O sistema permitirá aos administradores de excluirem cadastros de qualquer usuário;
- Os usuários do sistema somente poderão fazer uma denúncia mediante de uma justificativa plausível;
- O sistema terá um sistema de segurança que controla o que cada tipo de usuário do sistema pode acessar ou não;
- O sistema terá dois tipos de usuários os usuários comuns e os administradores;
- Somente os usuários "donos" de um serviço podem fazer uma edição ou uma exclusão do mesmo;
- Os usuários poderão alterar e exluir somente os seus respectivos cadastros;
- O sistema fará um relatório a cada conclusão de uma operação feita por um usuário;
- Qualquer tipo de usuário poderá buscar, visualizar e cadastrar um serviço.




O sistema NET3 Services terá usuários e visitantes. Visitantes são pessoas não logadas no sistema, e os usuários por sua
vez são pessoas logadas no sistema.

O usuário do sistema poderá prestar ou solicitar um serviço,
para prestar um serviço ele terá que fornecer algumas
informações sobre o serviço que ele presta.

Legenda do diagrama de casos de uso do projeto de conclusao de curso 2011:

- Fazer denuncias: todo tipo de usuario pode fazer denuncias contra outros usuários, como usuario fake e outros;
- Nota do serviço: a cada serviço contratado o usuário contratante poderá dar opiniões sobre o serviço prestado e dar
                   nota sobre o serviço;
- Tipo do serviço: os serviços no NET3 Services são separados por categorias. Exemplos: serviços domesticos, serviços de limpeza
                   , serviços de advocacia, serviços de eletrica, conserto de aparelhos eletronicos etc.;
