Senha: senha que será utilizada para acesso ao site.


Dica: para sua segurança evite utilizar valores como data de nascimento como sua senha. Para ter uma senha mais segura utilize caracteres especiais e letras maiusculas e minusculas. Caso queira uma sugestão de senha segura pressione o botão "Gerar Senha" para obte-la.


O senha deve possuir:
  • 1) Entre 6 e 20 caracteres alfanuméricos
  • 2) Pode possuir qualquer sinal de pontuação
